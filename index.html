Langage de programmation php

Présentation du php:

Le langage php permet de créer des pages web dynamique en passant par un serveur. Pour écrire et lire du php, les programmes Apache, PHP et MySQL sont néccessaires. Il est possible d'obtenir un package de ces programmes en téléchargant mamp. les pages web contenant du php ont l'extension .php .

Les variables superglobales:

Variables superglobales Definition Exemple
$_GET Variable superglobales qui détermine un array (tableau associatif) dans lequel son stocké les valeurs qui transitent par l'url.Lien vers la documentation php. <? php
echo 'Bonjour' . htmlspecialchars($_GET["pseudo"]) .'!' ;
?>
$_POST Contient les informations envoyés par un formulaire. <? php
echo 'Bonjour' . htmlspecialchars($_POST["pseudo"]) .'!' ;
?>
$_SERVER Variable de serveur et d'éxecution. Se sont les valeurs renvoyés par le serveur. Lien vers la documentation php contenant toutes variables en relation avec le serveur.
$_SESSION Permet de créer plusieurs variables qui restent stockées sur le serveur le temps de la présence d'un visiteur. Il faut appeler "session_start()" avant le code html pour pouvoir accéder aux variables superglobales. Lien vers la documenation php de toutes les fonctions de Session. <? php
session_start();
/*la session est démarrée si vous n'écrivez pas cette ligne, vous ne pouvez pas utiliser la variable globale $_Session*/
$_SESSION ["newsession"] = $value ;
?>
$_COOKIE Contient les valeurs des cookies enregistrés sur l'ordinateur du visiteur. Cela nous permet de stocker des informations sur l'ordinateur du visiteur pendant plusieurs mois, pour se souvenir de son nom, par exemple. Une fois que les cookies ont été placés, ils seront accessible lors du prochain chargement de page dans le tableau $_COOKIE. Tout comme "session_start", "set_cookie" doit être placé en haut de page avant tout code html. Lien vers la documentation php. <? php
setcookie ('pseudo', 'Xavier', time() + 365 + 24 + 3600);
?>
$_FILES Variable de téléchargement de fichier via http. Contient la liste des fichiers qui ont été envoyés via un formulaire. Lien vers la documentation php conscernant la gestion des téléchargements de fichiers.

Base de données Mysql:

Afin de conserver et stocker de manière organisée les données écritent en php, il va falloir les enregistrer dans une base de données (bdd). La Base de données va permettre de classer les informations. Imaginons la structure d'une base de données comme un tableau appelé la "base" dans lequel l'entête serai appelés une table ou les données seraient enregistrés dans des colonnes appelé des champs et les lignes des entrées. Grace au langage sql on cummunique avec Mysql. Ce langage est commun à tous systèmes de gestion de base de données.
